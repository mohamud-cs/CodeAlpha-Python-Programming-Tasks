# CodeAlpha Python Programming Tasks
A collection of small Python projects completed as part of the CodeAlpha Python Programming Virtual Internship 
where I challenged myself to explore and build on my programming concepts with Python.

## Tasks Completed
### ✅ Task 1: Hangman Game
A simple text-based Hangman game where the player guesses a word one letter at a time.
How it works:

A random word is chosen from a list of 5 predefined words
The player guesses one letter at a time
The player has 6 incorrect guesses before the game is over

Concepts used: random, while loop, if-else, strings, lists
File: hangman_game.py

### ✅ Task 4: Basic Chatbot
A simple rule-based chatbot that responds to basic user inputs.
How it works:

The user types messages like "hello", "how are you", or "bye"
The chatbot replies with predefined responses

Concepts used: if-elif, functions, loops, input/output
File: chatbot.py
